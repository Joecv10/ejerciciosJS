Propiedad curiosa

Los números 46 y 96 tienen una curiosa propiedad su producto no se altera aunque las cifras que lo integran cambien de lugar:
46 x 96 = 4416
64 x 69 = 4416
Confeccione un programa para imprimir todos los números de dos cifras que cumplen con esta propiedad.

Números naturales

Confeccione un programa para hallar e imprimir los números naturales que cumplan simultáneamente las condiciones siguientes:
• Se encuentren entre 1200 y 2000.
• Tengan exactamente 4 divisores enteros positivos que son 1,5, otro número primo y el propio número.

Cadenas de texto

Realizar un programa en JS para leer cadenas de texto hasta que se introduzca la cadena vacía o mientras que el número de vocales sea menor que 50. Al finalizar se debe de mostrar el número de caracteres introducidos y el número de vocales. En el caso de que el usuario introduzca una palabra menor a tres caracteres, no se debe procesar y se llevará cuenta del número de palabras de este tipo introducidas. Al finalizar se debe de mostrar el número de palabras menor de tres caracteres introducidas e imprimir la cadena total leida al reves.
