Ejercicio 2

Escribe un programa jS que el usuario ingrese un número entero. A continuación muestra un mensaje indicando si el número es positivo o negativo, si es par o impar, si es múltiplo de 5, si es múltiplo de 10 y si es mayor o menor que 100. Consideraremos el 0 como positivo. Utiliza el operador condicional ( ? : )
Si por ejemplo C = 55 la salida será:
• 55 es positivo
• 55 es impar
• 55 es múltiplo de 5
• 55 no es múltiplo de 10
• 55 es menor que 100
